About This Game

This game is simple to play but I'm still very happy with it because it was the first original game I ever made. I created this game for a college project where we were assigned to create our own prototype based on what we learned. At the time this game had a single level, no tutorials, and not even a real title. While I have graduated since then I wish to continue working on this project, adding new levels and ironing out any problems for the time being.
